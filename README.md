# pH Sensor Extension for MicroBlock

Extension สำหรับอ่านค่า pH จาก Analog pH Sensor พร้อมฟังก์ชันคาลิเบรท

## Features
- อ่านค่า pH (0-14)
- อ่านค่าแรงดันจาก pH Sensor (0-3.3V)
- ตั้งค่า Offset สำหรับคาลิเบรท

## Supported Boards
- ESP32
- Arduino

## Installation
ติดตั้งผ่าน MicroBlock IDE Extension Manager

## Blocks
1. **อ่านค่า pH (ขา A)** - คืนค่า pH
2. **อ่านค่าแรงดัน pH (ขา A)** - คืนค่าแรงดัน (V)
3. **ตั้งค่า pH Offset** - ปรับค่าสำหรับคาลิเบรท
